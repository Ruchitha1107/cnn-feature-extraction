# CNN Feature Extraction Tutorial

## Overview
This project demonstrates how Convolutional Neural Networks (CNNs) extract features from images using the MNIST dataset.

## Features
- CNN model implementation
- Feature map visualisation
- Analysis of convolution layers

## Dataset
MNIST handwritten digits dataset

## How to Run
1. Install dependencies:
   pip install tensorflow matplotlib numpy

2. Run the notebook:
   cnn_feature_extraction.ipynb

## Results
The model successfully extracts features such as edges and shapes from images.
